# Flame ADE V2 🔥

AI-Native Terminal & Development Environment — workspace lengkap untuk mengembangkan aplikasi/situs dari nol hingga production dalam satu tempat.

## Tech Stack
- **Frontend**: React 19, TypeScript, Vite 8, Tailwind v4
- **Backend**: Rust, Tauri 2
- **AI**: opencode.ai, gemini-cli, openrouter
- **Terminal**: xterm.js, CodeMirror 6

## Development
```bash
pnpm install
pnpm tauri dev
```
