# V2 Mockup Design Skill

Guide untuk mengimplementasikan mockup `mock-up FlameV2.html` ke kode React.

## Referensi
File mockup: `mock-up FlameV2.html` (di root proyek)

## Cara Pakai
1. Buka mock-up FlameV2.html di browser atau editor
2. Identifikasi komponen yang akan dibuat
3. Buat komponen React + Tailwind v4 sesuai DESAIN.md token
4. Bandingkan hasil dengan mockup secara visual

## Komponen dalam Mockup
- Header (traffic lights, logo, tabs, AI pill, actions)
- Sidebar Rail (icon-only, 48px wide, active state with indicator)
- File Explorer (tree, search, folder/file icons, badges)
- Tab Bar (tabs with colored dot indicators)
- Terminal Window (block-based output, prompt, git status, cursor)
- AI Panel (3D sphere, particles, chat, slash commands, input)
- Approval Dialog (timer, file context, approve/reject)
- Context Menu (actions with keyboard shortcuts)
- Status Bar (branch, errors, warnings, tests, AI status)
